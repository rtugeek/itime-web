### 说明

全部开源组件仓库在这：https://github.com/widget-js/widgets

### 运行项目
#### 1.下载并运行桌面组件客户端
- https://www.microsoft.com/store/productId/9NPR50GQ7T53
- 或
- https://widgetjs.cn

#### 2.克隆代码后运行
```shell
# 先安装依赖
pnpm install
# 再执行
pnpm run dev
```
